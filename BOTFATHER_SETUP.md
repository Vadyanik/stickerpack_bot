# Настройка бота через BotFather

## ВАЖНО! Настройка Privacy Mode

Чтобы бот мог видеть все сообщения (включая стикеры) в группе, нужно **отключить Privacy Mode**.

### Шаги:

1. **Откройте [@BotFather](https://t.me/botfather)** в Telegram

2. **Отправьте команду** `/mybots`

3. **Выберите вашего бота** из списка

4. **Нажмите** `Bot Settings`

5. **Нажмите** `Group Privacy`

6. **Нажмите** `Turn off` (или `Disable`)

Должно появиться сообщение:
```
Privacy mode is disabled for @ваш_бот.
```

### Что это делает:

- **Privacy Mode ON** (включен) - бот видит только:
  - Команды (начинаются с `/`)
  - Сообщения с упоминанием бота
  - Сообщения в ответ на сообщения бота

- **Privacy Mode OFF** (выключен) - бот видит:
  - ✅ ВСЕ сообщения в группе
  - ✅ Все стикеры
  - ✅ Все медиафайлы

### После отключения Privacy Mode:

1. **Удалите бота** из группы
2. **Добавьте бота** обратно в группу
3. Отправьте `/start` чтобы проверить
4. Отправьте стикер - бот должен его обработать

## Дополнительные настройки (опционально)

### Установить описание бота:

```
/setdescription - краткое описание бота
```

Пример:
```
Автоматически сохраняю все стикеры из группы в общий стикерпак.
Поддерживаю обычные, анимированные и видео стикеры!
```

### Установить команды бота:

```
/setcommands
```

Затем отправьте:
```
start - Информация о боте
help - Справка по использованию
```

### Установить About текст:

```
/setabouttext
```

Пример:
```
Бот для автоматического сохранения стикеров в стикерпак
```

## Проверка настроек

После всех настроек:

1. Перезапустите бота
2. Удалите бота из группы
3. Добавьте обратно
4. Отправьте `/start` в группу
5. Отправьте стикер
6. Бот должен создать пак и отправить ссылку

## Если всё равно не работает

Проверьте логи бота в консоли:
- Должны быть сообщения "Получен стикер | Тип чата: group..."
- Если таких сообщений нет - Privacy Mode всё ещё включен
