# Быстрый старт

## NixOS / Ubuntu / Любая Linux система

### 1. Установите зависимости

**Автоматическая установка (рекомендуется):**
```bash
./setup.sh
```

**Ручная установка:**
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Получите токен бота
1. Откройте [@BotFather](https://t.me/botfather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

### 3. Настройте токен
```bash
# Скопируйте пример
cp .env.example .env

# Отредактируйте .env и вставьте свой токен
nano .env
# или
vim .env
```

### 4. Запустите бота

**С виртуальным окружением (NixOS и другие):**
```bash
source venv/bin/activate
python3 bot.py
```

**Без активации venv:**
```bash
venv/bin/python bot.py
```

**Через start.py (определит venv автоматически):**
```bash
./start.py
```

**На Ubuntu (если установлено глобально):**
```bash
./run.sh
```

### 5. Добавьте бота в группу
1. Добавьте бота в вашу Telegram группу
2. Отправьте любой стикер
3. Бот создаст стикерпак и отправит ссылку

## Готово!

Теперь все стикеры из группы будут автоматически сохраняться в стикерпак.

## Устранение неполадок

**Ошибка "bad interpreter":**
- Используйте `./start.py` вместо `./run.sh`

**Ошибка "BOT_TOKEN not set":**
- Проверьте файл `.env`
- Или установите: `export BOT_TOKEN="ваш_токен"`

**Бот не отвечает:**
- Проверьте, что бот запущен
- Проверьте логи в консоли
- Убедитесь, что бот добавлен в группу
